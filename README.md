### Cube event emitter written in bash

```
usage: ./cube-emitter [options] TYPE [PAYLOAD]

Emit an event to cube

You need to provide a TYPE. PAYLOAD is optional, but must be in JSON format.

OPTIONS:
   -h      show this message
   -m      id of the message
   -c      Cube server host
   -p      Cube server port
   -t      time of the message
```
